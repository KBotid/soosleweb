# 🌐 Soosle Web 1.0

Soosle Web 1.0 is a custom search engine homepage powered by Google Programmable Search (CSE).  
It is designed as a simple, fast hub for finding coding websites, AI tools, video editors, photo editors, game development tools, and creator resources.

---

## 🚀 Features

- 🔍 Google CSE-powered search
- 💻 Coding & development websites
- 🤖 AI tools search
- 🎬 Video editing tools
- 🖼️ Photo editing tools
- 🎨 Design platforms
- 🎮 Game development resources
- ⚡ Lightweight and fast webpage
